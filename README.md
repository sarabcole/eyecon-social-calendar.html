# Eyecon Social Media Calendar Generator

Internal tool for generating on-brand social media content for Eyecon.

## What it does
Generates monthly social media calendars and one-off posts for Facebook and Instagram, 
tailored by market (Pakistan, India, Bangladesh, US) and automatically aligned to 
Eyecon's brand voice, messaging pillars, and core features.

## How to use
1. Open the tool at: https://sarabcole.github.io/eyecon-social-calendar.html/
2. Paste your Anthropic API key
3. Select your settings and generate
